Python Şifre Kırma Simülasyonu

Amaç

Kullanıcının belirlediği şifreyi, olası tüm kombinasyonları (en düşük karakterden en yükseğe doğru) hızlıca deneyerek bulmaya çalışan bir **Brute-Force algoritmasının** çalışma mantığını göstermek. Deneme hızı, kullanılan bilgisayarın işlem gücüne bağlıdır.

Video

Çalışma Mantığı

Bu projede, Python'da `random` kütüphanesi kullanılarak deneme karakterleri (harf ve rakamlar) tanımlanmıştır. Kod, belirlenen şifre uzunluğunda rastgele karakterler üretir ve şifre bulunana kadar denemeye devam eder. Bu simülasyon, kısa ve zayıf şifrelerin ne kadar hızlı bulunabileceğini göstermektedir.

Kaynak Kod (Python)


# BURAYI DEĞİŞTİR: Python kodunu buraya yapıştır.
from random import randint

KEYS = [
    "1", "2", "3", "4", "5", "6", "7", "8", "9", "0",
    "a", "b", "c", "d", "e", "f", "g", "h", "i", "j",
    "k", "l", "m", "n", "o", "p", "q", "r", "s", "t",
    "u", "v", "w", "x", "y", "z"
]


def brute_force(password: str) -> None:
    pwg = ""
    while pwg != password:
        pwg = ""
        for _ in range(len(password)):
            guess_pass = KEYS[randint(0, len(KEYS) - 1)]
            pwg = f"{guess_pass}{pwg}"
            print(f"\rattacking... please wait! {pwg}", end="", flush=True)
    print()  # newline after carriage-return loop
    print(f"The pass is: {pwg}")


def main() -> None:
    while True:
        password = input("Send the password (max 4 chars): ")
        if len(password) == 0:
            print("Boş şifre kabul edilmiyor.")
            continue
        if len(password) > 4:
            print("En fazla 4 karakter deneyebilirsin. Lütfen yeniden dene.")
            continue
        brute_force(password)
        again = input("Tekrar denemek ister misin? (yes/no): ").strip().lower()
        if again not in ("y", "yes"):
            print("Görüşürüz!")
            break


if __name__ == "__main__":
    main()

Potansiyometre ile Servo Kontrolü

Amaç

Arduino'nun Analog Giriş (Analog Read) özelliğini kullanarak bir Potansiyometre'den gelen voltaj değerini okumak ve bu değeri Servo Motor'un hareket açısına (0-180 derece) dönüştürerek motoru hassasça kontrol etmektir. Bu, C++ (Arduino) ile donanım kontrolünün temelidir.

Video

Devre Şeması

Potansiyometrenin orta pini **Arduino A0** analog girişine, yan pinleri **5V ve GND**'ye bağlanmıştır. Servo Motor'un veri (sarı) pini **Arduino Dijital 9**'a, güç ve toprak pinleri uygun şekilde bağlanmıştır.

Kaynak Kod (C++)


// BURAYI DEĞİŞTİR: Arduino (C++) kodunu buraya yapıştır.
#include 

Servo myServo;  // Servo nesnesi oluşturuldu
int potPin = A0;  // Potansiyometre analog giriş pini
int val;      // Potansiyometre değeri için değişken

void setup() {
  myServo.attach(9); // Servo sinyal pini 9'a bağlandı
}

void loop() {
  // 1. Potansiyometreden 0-1023 arası değer oku
  val = analogRead(potPin); 
  
  // 2. 0-1023 değerini Servo'nun 0-180 derecesine dönüştür
  val = map(val, 0, 1023, 0, 180);
  
  // 3. Servo'yu yeni açısına hareket ettir
  myServo.write(val);
  
  delay(15); 
}
