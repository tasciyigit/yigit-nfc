Arduino ile LED Kontrolü

Projenin Amacı

Bu projenin amacı, mikrodenetleyici dünyasına giriş yaparak dijital pinlerin kontrol mantığını kavramaktır. 5V çıkış vererek bir LED'i belirli aralıklarla yakıp söndürmeyi hedefler.

Devre Şeması

Kaynak Kod


// Arduino LED Yakma Kodu
void setup() {
  pinMode(13, OUTPUT); // 13. Pini çıkış olarak ayarla
}

void loop() {
  digitalWrite(13, HIGH); // LED'i yak
  delay(1000);            // 1 saniye bekle
  digitalWrite(13, LOW);  // LED'i söndür
  delay(1000);            // 1 saniye bekle
}
